## New Features

- **[CI/CD Workflows]**: Added a new GitHub Actions workflow (`tests-docs.yml`) to automatically test the Hugo documentation build process. This workflow ensures that the documentation builds successfully on pushes, pull requests, and related issue activities. It leverages Hugo version `0.156.0` (extended) and Go `1.26` for the build process.
