## New Features

- **[Python Example Module]**: Updated settings keys to use a standardized `example-*` naming convention (e.g., `example-port`, `example-string`, `example-bool`) to clearly differentiate them from built-in ASLM system variables.
- **[Python Example Module]**: Refined the description text for all settings to explicitly detail how ASLM passes these values to the module and normalizes types, improving clarity for module authors.

## Bug Fixes

- **[Python Example Module]**: Removed `Settings/host_theme.json` and `Settings/host_locale.json` from the `preserve` array. These files are regenerated by the host and do not need to be preserved across module updates.
