## File `GlobalUsings.cs`

Central **`global using`** directives for the MAUI installer project (`Installer/Installer/GlobalUsings.cs`). Keeps XAML code-behind and UI code free of repetitive `Microsoft.Maui.*` imports.

No runtime behavior — edit only when adding MAUI namespaces used across multiple installer files.
