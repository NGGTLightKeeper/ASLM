## Class `LocalizationKeys`

`ASLM/Localization/LocalizationKeys.cs` — **`public static`** — compile-time **`const string`** names for RESX keys (mirrors [AppResources.resx](../Resources/Strings/)).

Marked `// <auto-generated />` in source; regenerate when RESX entries are added or renamed.

**No methods** — only `public const string` members (one per resource key).

---

## Naming convention

| Prefix | Typical use |
| --- | --- |
| `AppShell_*` | Shell navigation and title |
| `Common_*` | Shared buttons and labels |
| `Consoles_*` | Consoles page |
| `Modules_*` | Modules page |
| `Settings_*` | Settings and personalization |
| `Notifications_*` | Notification center |

---

## Usage

```csharp
L.Get(LocalizationKeys.AppShell_Title)
```

Prefer constants over string literals so renames are caught at compile time.
