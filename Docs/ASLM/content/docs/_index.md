**ASLM** (**Advanced System Language Models**) is a modular Windows desktop host built with **.NET MAUI**. It loads **modules** and **engines** from on-disk manifests (installed and updated according to user configuration), coordinates downloads and updates, and exposes **module interop** and a local **ASLM API** to embedded module UIs.

Distribution is handled by **Launcher**, **Patcher**, and **Installer** alongside the main `ASLM` application.

This site documents the **ASLM solution codebase** (C#) and **patch notes**.
